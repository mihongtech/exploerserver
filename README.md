# linkchain-explorer

## Installation (Linux or Mac)

Build program

`./build.sh`

Run

`lcexplorer`

Run nohup

`nohup lcexplorer >/**/*/debug.log 2>&1 &`

Then the server running at `http://127.0.0.1:9100`

## Installation (Windows)

Build program

`./build.bat`

Run (Windows)

`lcexplorer.exe`

Then the server running at `http://127.0.0.1:9100`
